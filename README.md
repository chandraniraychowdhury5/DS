 Data Science Portfolio

This repository contains my data science and machine learning projects focused on renewable energy, electricity price analysis, forecasting, and statistical data analysis. The projects demonstrate the complete data science workflow, including data preprocessing, exploratory data analysis (EDA), statistical analysis, feature engineering, machine learning, and predictive modeling.
